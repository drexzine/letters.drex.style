{%- comment -%} byline.html — renders the author chip for a post. Looks up `post.author` (the front-matter key) in _data/authors.yml and renders a photo + name + role chip linking to /authors//. If the post has no `author` key (or it isn't in authors.yml), it falls back to site.author as a plain text name — no broken link, no missing photo. API: {% include byline.html author=page.author %} {%- endcomment -%} {%- assign key = include.author | default: page.author -%} {%- assign a = site.data.authors[key] -%} {%- if a -%} {%- assign accent = a.accent | default: 'grass' -%} {%- if a.photo %}{%- endif %} {{ a.name }} {%- if a.role %}{{ a.role }}{%- endif %} {%- else -%} {{ page.author | default: site.author }} {%- endif -%}
